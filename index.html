About

Glitch Gobblers is a 2D maze game with a Viking twist. You wake inside a fortress-like maze and need to find your way out before the clock runs down. The goal is simple: escape — but the path is not. The map blends university-like spaces with winding corridors and tight routes.

There are obstacles, both visible and hidden: locked paths, spiked floors, and secret passages that only appear when you’re paying attention (or just lucky). Not all encounters slow you down either — some items help you move faster or survive the next challenge.

The timer matters. The default run is five minutes, and difficulty levels adjust the time to raise or lower the pressure. The game starts paused and can be paused at any moment, keeping the experience casual and fair.

You win by escaping and making it “home” after finding your way out. Sprint straight for the exit, or explore for helpful boosts and risk a longer route. Beat the clock, outsmart the maze, and prove you’re not meant to wander university halls forever.

Game & Setup

Download the executable JAR and follow the steps below.

  1. Download: GlitchGobblers.jar
  2. Ensure you have Java 17+ installed.
  3. Run from terminal: java -jar GlitchGobblers.jar

Note: Put your JAR into the /docs folder in the repo so this link works on GitHub Pages.
